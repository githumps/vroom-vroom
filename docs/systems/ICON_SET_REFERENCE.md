**Version:** 2.0.0

## Overview

This document defines all 120 icons for VROOM VROOM. Each icon is designed as pixel-perfect 16x16px art following the cozy cyberpunk aesthetic.

---

## Icon Specifications

### Technical Requirements

- **Size**: 16x16 pixels
- **Grid**: 4x4 pixel base unit
- **Style**: Isometric where applicable, flat for UI elements
- **Color Palette**: Use design tokens (see visual-system.css)
- **Rendering**: `image-rendering: pixelated`
- **Format**: PNG with transparency
- **Sprite Sheet**: 160px × 192px (10 columns × 12 rows)

### Design Guidelines

1. **Clarity**: Icons must be readable at 16x16px
2. **Consistency**: Use same perspective across related icons
3. **Color**: Max 4 colors per icon (including transparency)
4. **Silhouette**: Strong, recognizable shapes
5. **Detail**: Minimal but expressive
6. **Glow**: Optional 1-2px glow for emphasis

## CSS Implementation

```css

    width: 16px;
    height: 16px;
    display: inline-block;
```

## Design Process

### 1. Sketch Phase
- Draw 16x16 grid on paper
- Rough out icon concept
- Test at 100% size (actual 16px)

### 2. Pixel Phase
- Use pixel art software (Aseprite, Pixaki, Photoshop)
- Place on 16x16 canvas
- Use 4x4 grid for alignment
- Limit to 4 colors max

### 3. Review Phase
- Export at 1x size
- View at actual screen size
- Check readability
- Adjust if unclear

### 4. Sprite Sheet Phase
- Arrange in 10×12 grid
- Maintain exact spacing
- Export as PNG
- Test CSS positioning

## Accessibility

### Best Practices

1. **Always pair icons with text labels** (or tooltips)
2. **Use ARIA labels** for icon-only buttons
3. **Maintain contrast** (4.5:1 minimum)
4. **Provide alt text** when appropriate
5. **Test with screen readers**

### Example

```html
<!-- Good: Icon + text -->
<button class="btn btn-primary">
    <span class="icon icon-save" aria-hidden="true"></span>
    Save Game
</button>

<!-- Good: Icon-only with label -->
<button class="btn btn-ghost" aria-label="Close modal">
    <span class="icon icon-close" aria-hidden="true"></span>
</button>

<!-- Bad: Icon-only without label -->
<button class="btn">
    <span class="icon icon-close"></span>
</button>
```

## Production Notes

### File Delivery

1. **Source Files**: Aseprite (.ase) or Photoshop (.psd) with layers
2. **Sprite Sheet**: PNG at 160×192px (1x)
3. **High-DPI**: PNG at 320×384px (2x) and 480×576px (3x)
4. **CSS File**: `icons.css` with all positioning
5. **Documentation**: This reference file

### File Naming

```
icons-spritesheet-1x.png (160×192px)
icons-spritesheet-2x.png (320×384px)
icons-spritesheet-3x.png (480×576px)
icons.css (CSS classes)

```
