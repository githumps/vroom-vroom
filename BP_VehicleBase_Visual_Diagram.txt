================================================================================
PIN CONNECTION GUIDE
================================================================================

EXECUTION PINS (White):
  → Connect: Event Tick → Get Input Axis nodes (all 3 chains)
  → Connect: Get Input (Brake) → Branch node
  → Connect: Branch TRUE → Get Velocity

DATA PINS:
  Float (Green):
    → Axis Value → Multiply nodes
    → Multiply results → Vector*Float nodes

  Vector (Yellow):
    → GetActorForward/Up → Vector*Float nodes
    → GetVelocity → Vector*Float
    → Vector*Float results → Add Force/Torque nodes

  Boolean (Red):
    → Float comparison → Branch Condition

================================================================================
NODE SEARCH TERMS (for Blueprint Editor)
================================================================================

Type these EXACTLY in the node search box:

  "Get Input Axis Value"       → Input system
  "float * float"              → Multiply float by float
  "Get Actor Forward Vector"   → Self reference
  "Get Actor Up Vector"        → Self reference
  "Vector * Float"             → Multiply vector by scalar
  "Add Force"                  → Physics force (linear)
  "Add Torque in Radians"      → Physics torque (rotational)
  "float > float"              → Greater than comparison
  "Branch"                     → Conditional execution
  "Get Velocity"               → Current movement velocity

================================================================================
COMMON MISTAKES TO AVOID
================================================================================

1. ✗ Forgetting to set Axis Names in Get Input Axis Value nodes
   ✓ Double-check all three: MoveForward, MoveRight, Brake

2. ✗ Not checking "Velocity Change" on Add Force/Torque nodes
   ✓ ALL force/torque nodes must have this enabled!

3. ✗ Using wrong multiply values (1500 vs 15000, etc.)
   ✓ Double-check each value matches specification

4. ✗ Connecting execution pins to data pins
   ✓ White→White, Green→Green, Yellow→Yellow

5. ✗ Missing the Branch node in braking chain
   ✓ Brake MUST use Branch to prevent constant force application
