================================================================================
EVIL VALIDATION LOGIC (ALREADY IN C++)
================================================================================

IF ANY FIELD EMPTY OR ANY CHECKBOX UNCHECKED:
  1. Show "INCOMPLETE FORM! START OVER!" message
  2. Play error sound
  3. Wait 2 seconds (comedic timing - player realizes doom)
  4. Clear ALL fields and uncheck ALL checkboxes (nuclear option)
  5. Hide error message

IF ALL FIELDS FILLED AND ALL CHECKBOXES CHECKED:
  1. Play submit sound
  2. Call ShowSentence() Blueprint event
  3. Wait 3 seconds
  4. Call ReturnToOpenWorld() Blueprint event
  5. Close form

================================================================================
BLUEPRINT EVENTS TO IMPLEMENT
================================================================================

Event: ShowSentence
  → Create WBP_SentenceDisplay widget
  → Add to viewport
  → Play reveal animation

Event: ReturnToOpenWorld
  → Get Player Controller
  → Cast to BP_VroomPlayerController
  → Call ReturnToOpenWorld function
  → Remove widget from parent

Event: PlayErrorSound
  → Play Sound 2D: ErrorBuzzer_Cue

Event: PlaySubmitSound
  → Play Sound 2D: FormStamp_Cue
